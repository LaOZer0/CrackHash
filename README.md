# CrackHash

Проект для взлома/подбора хешей.

## Описание

CrackHash — инструмент для работы с хешами.

## Требования

- .NET SDK (последняя версия)

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd CrackHash
   ```

2. Восстановите зависимости:
   ```bash
   dotnet restore
   ```

## Использование

```bash
dotnet run
```
